About Me

After spending 5 years in the finance world I decided to take a break to become a caregiver. As I prepared to re-enter the workforce, I came up with a game plan to pursue a career in technology.

My passion for creating led me down the path of Front-End Web Development. While participating in various online training programs to learn web development, I acquired the skills to complete various front-end and full-stack projects. In doing this training, I was also led down the path of Cloud technology and Devops. Realizing that this was going to be a much needed skill going forward into my new career path, I obtained skills in all major cloud Platforms. On this journey I have built projects and obtained certifications on the AWS and Azure platforms.

I'm currently honing my Python skills, building projects, and pursuing Cloud certifications. I currently hold the Azure Fundamentals (AZ-900), Azure Data Fundamentals (DP-900), Microsoft Azure Administrator - Associate (AZ-104), AWS Certified Cloud Practitioner, and the AWS Solutions Architect - Associate certifications. I was awarded training for the Google IT Automation with Python professional certificate on Coursera from the Blacks in Technology Foundation and selected for Google's ACE Jumpstart Program to obtain the GCP Associate Cloud Engineer certification. Time Start Building on GCP!

My passion for creativity, problem solving, and learning new tools to enhance my development skills, is a driving factor in my life. But when the opportunity arises, I enjoy time on the beach feeling the breeze and listening to the waves.

AWS Connect Vanity

In this project I created a Lambda that converted a caller's phone number to vanity numbers when they phoned into an Amazon Connect Contact Center. The best 5 vanity numbers along with the caller's number will be saved in a DynamoDB table. An Amazon Connect contact flow will be created that repeats the caller's phone number and vanity possibilities that the Lambda function returns.

Python, Docker, Git and GitHub, Amazon Connect, AWS Lambda, DynamoDB, SAM CLI
