AWS



AWS Connect Vanity

The purpose of this project was to create a Lambda that converts a caller's phone number to vanity numbers when they call into an Amazon Connect Contact Center. The best 5 resulting vanity numbers along with the caller's number will be saved in a DynamoDB table. An Amazon Connect contact flow will be created that looks at the caller's phone number and repeats the vanity possibilities that come back from the Lambda function.


Python, Docker, Git and GitHub, Amazon Connect, AWS Lambda, DynamoDB, SAM CLI
